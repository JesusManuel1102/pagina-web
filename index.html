Bienvenido al curso de HTML y CSS

En este curso aprenderás los fundamentos de HTML y CSS para crear páginas web atractivas y funcionales.

Temario

  • Introducción a HTML: Aprende los conceptos básicos de HTML, el lenguaje de marcado estándar para crear páginas web.
  • Introducción a CSS: Explora los fundamentos de CSS, el lenguaje de estilos utilizado para dar formato y diseño a las páginas web.
  • HTML semántico: Comprende la importancia del HTML semántico para mejorar la accesibilidad y la estructura de tus páginas web.
  • Modelo de caja en CSS: Domina el modelo de caja CSS, que describe cómo se renderizan los elementos HTML en una página web.
  • Diseño responsivo: Aprende a crear diseños web que se adapten y funcionen correctamente en una variedad de dispositivos y tamaños de pantalla.
